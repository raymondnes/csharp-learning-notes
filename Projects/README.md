## How to Use

### Console Projects (01-60)
```bash
dotnet new console -n ProjectName
cd ProjectName
# Copy starter code to Program.cs
dotnet run
```

### API Projects (61-80)
```bash
dotnet new web -n ProjectName
cd ProjectName
# Follow project instructions
dotnet run
```

## Recommended Progression

1. **Week 1-2:** Projects 01-10 (Basic syntax)
2. **Week 3-4:** Projects 11-20 (Methods, loops)
3. **Week 5-6:** Projects 21-30 (Classes, OOP)
4. **Week 7-8:** Projects 31-40 (Error handling, files)
5. **Week 9-10:** Projects 41-50 (Inheritance, interfaces)
6. **Week 11-12:** Projects 51-60 (LINQ, patterns)
7. **Week 13-14:** Projects 61-70 (APIs, EF Core)
8. **Week 15-16:** Projects 71-80 (Full applications)

## Tips for Success

1. **Try First** - Attempt each project before looking at hints
2. **Test Often** - Run your code frequently
3. **Read Errors** - Error messages are helpful
4. **Refactor** - Improve your code after it works
5. **Document** - Add comments explaining complex logic
